Everything you need to track markets

Built natively for macOS with performance and simplicity in mind.

Real-time Prices

Live price updates from Binance API for crypto and Yahoo Finance for gold & indices. No delays, always current.

Multi-Asset Support

Track 8+ cryptocurrencies plus Gold (XAUUSD), Dow Jones (US30), and Nasdaq 100 (NAS100) all in one place.

Price Alerts

Set custom price targets and get native macOS notifications when your assets hit the mark.

Menu Bar Display

Choose up to 3 assets to display simultaneously in your menu bar with live prices and 24h change indicators.

Auto-Refresh

Automatic price updates every 15 seconds with smart rate limiting to prevent API throttling.

Lightweight

Built with Python and rumps. No Electron bloat. Minimal memory and CPU usage.

Up and running in seconds

No accounts, no API keys, no setup. Just download and go.
